# Email Automation with Python

This project sends motivational quotes,Daily DSA problem and a python pro tip, via Gmail to a group of my friends. It supports sending emails either individually or to groups.
Making it useful for sharing information for a group of people automatically.

## Features
- Reads quotes,problems and tips from a text files
- Sends emails to multiple people
- Credentials stored securely in `.env`
- Uses Gmail SMTP with TLS encryption

## Setup
1. Clone the repo
2. Install dependencies:
open terminal and type:
   pip install -r requirements.txt
